# Quizzer Server

### Prerequisites
* NodeJS
* Docker
* NPM or YARN

### How to start
1. Install the dependencies

    ```yarn install```
    or
    ```npm install```

2. Start the Database Server

    ```yarn start:db```
    or
    ```npm run start:db```

3. Start the Server

    ```yarn start:server```
    or
    ```npm run start:server```
